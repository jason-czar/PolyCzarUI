PolyCzar Chrome Extension Guide

Note: This guide will help you load your PolyCzar application as a Chrome extension.

Step 1: Create Icon Files

Before loading the extension, you need to create icon files in the following sizes:

  • 16x16 pixels (icon16.png)
  • 48x48 pixels (icon48.png)
  • 128x128 pixels (icon128.png)

Place these files in the /public/assets/icon/ directory.

You can use any image editing software to create these icons with your PolyCzar branding.

Step 2: Build Your Application

Run the following command to build your React application:

pnpm build

This will create a production build in the dist directory.

Step 3: Copy Manifest to Dist

Make sure your manifest.json file is copied to the dist directory during the build process. If it's not automatically copied, run:

cp public/manifest.json dist/

Also ensure your icon files are copied:

mkdir -p dist/assets/icon
cp public/assets/icon/*.png dist/assets/icon/

Step 4: Load the Extension in Chrome

  1. Open Chrome and navigate to chrome://extensions
  2. Enable "Developer mode" by toggling the switch in the top-right corner
  3. Click "Load unpacked" button
  4. Select the dist directory of your PolyCzar application

Your PolyCzar extension should now appear in the extensions list and be available from the Chrome toolbar.

Step 5: Test Your Extension

Click on the PolyCzar icon in your Chrome toolbar to open the extension popup.

The extension should display your React application in the popup window.

Troubleshooting

Extension Not Loading

If your extension doesn't load properly:

  • Check Chrome's developer console for errors
  • Ensure all paths in manifest.json are correct
  • Make sure your build process is correctly copying all necessary files to the dist directory

Content Security Policy Issues

If you encounter CSP (Content Security Policy) issues, you may need to modify your manifest.json and/or adjust your application's code to comply with Chrome's extension security policies.

Publishing Your Extension

When you're ready to publish your extension to the Chrome Web Store:

  1. Create a ZIP file of your dist directory
  2. Create a developer account on the Chrome Web Store
  3. Follow the submission process on the Chrome Web Store Developer Dashboard

For more information, visit the Chrome Web Store publishing documentation.

Remember: Chrome extensions have certain limitations and security restrictions that may affect how your application functions. You may need to make adjustments to your code to ensure it works properly as an extension.
